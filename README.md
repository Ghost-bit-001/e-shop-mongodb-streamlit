# E-Shop Brasil - Gestão de Dados com MongoDB e Streamlit

Este projeto simula a gestão de dados de uma plataforma de e-commerce usando:

- MongoDB
- Docker
- Streamlit

## Tecnologias Usadas
- Docker
- MongoDB
- Streamlit
- Python

## Funcionalidades
- Inserção de clientes
- Visualização dos dados
- Edição/Exclusão (mínimo)
